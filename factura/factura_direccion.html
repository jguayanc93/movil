📋 Componentes de una Dirección Completa

1. Calle

Av. Principal, Calle 5, etc.

2. Número

123, 456 A, 789 B, etc.

3. Piso/Depto

Piso 5, Depto 12, etc. (opcional)

4. Localidad

Ñuñoa, La Reina, Maipú, etc.

5. Región

RM, Valparaíso, etc.

6. Referencias

Cercano a..., entre calles...

📝 Ejemplos de Direcciones

Completa: Av. Apoquindo 5678, Piso 3, Las Condes, Región Metropolitana

Simplificada: Calle Matta 123, Ñuñoa, RM

Con referencia: Pasaje 5 No. 45, Depto 8, entre Merced y Teatinos, Centro

📝 Resumen de Pasos

  1. 1. Ingresa el número de factura en el primer campo
  2. 2. Haz clic en el botón "Buscar" para cargar los datos
  3. 3. Verifica la dirección actual en la sección "Dirección Actual"
  4. 4. En "Asignar Nueva Dirección", ingresa la dirección completa y correcta
  5. 5. Asegúrate de incluir todos los componentes (calle, número, ciudad, región)
  6. 6. Los cambios se guardarán automáticamente ✓
